My Portfolio

Medium: HTML, CSS, Notepad++

This project is my portfolio, which is the website you are currently viewing.

I have decided to make my portfolio with code instead of using a website builder.
It is mostly made using HTML along with some CSS. While using a website builder
provides more ease and has more facility to convey an aesthetic layout, I wanted
to make it myself. I already know basic HTML and CSS from a website class I took
in college.

Before beginning, I decided on a few key features. I wanted a menu on the top of
the pages, a gallery with clickable images and an about me section. The colors changed
a bit with time, but I do not want to put overwhelming colors, nor too many. It is important
that the overall design of my portfolio does not take away from the selected pieces.

The portfolio went through a few vast changes. The changes included layout as well as
color. But the overall minimalistic simple look stayed.

The intention was to communicate an easy to navigate layout with all my works in it.
It has a project page with all projects, all in gallery format. Each project has its own
page. There is also an about me page.



This is the first prototype version. As noticed, the colors and navigation bar have changed.



This is the final prototype. The navigation bar is now more suitable, and any unnecessary titles
were removed. The design seems more full and fluid, which is preferred. It is also the stage where
I added a drop down menu.
